# GOIA Welcome Video

Place the provided GOIA welcome video here using one of these names:

- `goia-welcome.mp4`
- `goia-welcome.webm`

The homepage video automatically tries these local files first, then falls back to the bundled remote lounge video if they are not present.

Place the official transparent GOIA logo here as:

- `goia-logo-transparent.png`

The app uses this transparent logo on the cinematic loading and welcome screens.
